# Variables

ngrok provides a set of variables that you can use in Traffic Policy to:

- **Conditionally handle traffic** through [Expressions](/docs/traffic-policy/concepts/expressions).
- **Dynamically configure actions** using [CEL interpolation](/docs/traffic-policy/concepts/cel-interpolation).
- **Debug traffic flows** via actions like [`log`](/docs/traffic-policy/actions/log) or [`custom-response`](/docs/traffic-policy/actions/custom-response).

### Available Variables:

- [Action Variables](/docs/traffic-policy/action)
- [Connection Variables](/docs/traffic-policy/connection)
  - [GeoIP Variables](/docs/traffic-policy/connection#conngeo)
  - [TLS Variables](/docs/traffic-policy/connection#conntls)
- [Endpoint Variables](/docs/traffic-policy/endpoint)
- [Request Variables](/docs/traffic-policy/req)
- [Response Variables](/docs/traffic-policy/res)
- [Time Variables](/docs/traffic-policy/time)
